Descripción

Curso totalmente practico, enfocado en el manejo y aplicación de las diferentes herramientas de Excel, para desarrollar de manera más eficiente actividades relacionadas con la gestión, análisis de datos y presentación de reportes.

Objetivo general

En este curso aprenderás a construir funciones anidadas para generar reportes y gráficos específicos para el análisis de datos y aplicar eficientemente, en tu centro de trabajo, las herramientas de Excel para organizar, analizar, administrar, compartir información y automatizar la generación de reportes.


Requisitos

  • El participante debe contar con una computadora con: Micrófono, Cámara y Excel, preferentemente con la versión 2016 o posterior.
  • Es deseable haber cursado Excel Básico. En caso contrario el participante será responsable de su nivelación, con la finalidad de aprovechar al máximo el curso.
  • Conocimientos básicos del uso de una computadora y de diversas herramientas en la web.
  • Es deseable que cuente con habilidades de comunicación mediante diversas herramientas en la web, como el correo electrónico; administración del tiempo, autorregulación del aprendizaje, apertura al cambio y disposición de compartir conocimiento.

Contenido


  • Aspectos generales de Excel.
  • Atajos y uso de teclado para el manejo de archivos grandes.

  • Aplicación de las fórmulas más comunes.
  • Uso de formatos de celda para resaltar y centrar información.

  • La gestión de bases de datos para generar de forma ágil la gestión de reportes.
  • Aplicación de algunas funciones condicionales para la generación de reportes.

  • Anidamiento o concatenación de funciones para la obtención de resultados específicos.
  • Gestión de bases de datos con el uso de las funciones lógicas.

  • Anidamiento o concatenación de funciones para la obtención de resultados específicos.
  • Construcción de funciones anidadas para casos específicos.
  • Gestión de bases de datos con el uso de funciones lógicas.

  • Construcción de tablas a partir de rangos de información.
  • El uso y aplicación de los filtros.
  • Uso de formularios para la introducción de datos y la inclusión de nuevos elementos en la cinta de opciones.

  • Introducción del manejo de las tablas dinámicas.
  • Sus diferentes presentaciones a partir de la combinación de datos.
  • La gestión de bases de datos mediante el uso de las tablas dinámicas.
  • Los diferentes formatos y combinaciones de los campos de información.

  • Referencias relativas, absolutas y mixtas, para la generación de reportes y la gestión de bases de datos.

  • El uso de texto en columnas y el uso de formatos para la presentación de informes.
  • El uso de los formatos condicionales y la separación de texto de una celda para la gestión de bases de datos en forma eficiente.

  • El uso de texto en columnas y la concatenación de datos de diferentes celdas para la gestión de texto.
  • Construcción de fórmulas con funciones anidadas.
