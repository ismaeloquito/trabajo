Descripción:

Curso totalmente práctico enfocado al manejo y aplicación de las diferentes herramientas de Excel para desarrollar de manera más eficiente actividades relacionadas con la gestión, análisis de datos y presentación de reportes.

Objetivo general:

Formar conocimientos que permitan a los participantes una gestión más eficiente en el manejo de información y su análisis. A través de la adquisición de habilidades para desarrollar de manera más eficiente actividades relacionadas con la gestión, análisis de datos y presentación de reportes, dentro de sus centros de trabajo.

Requisitos
  • Uso básico de la computadora.
  • El participante debe contar con una computadora con: micrófono, cámara y Excel, preferentemente con la versión 2016 o posterior.
  • Deseable, autorregulación del aprendizaje, administración del tiempo, apertura al cambio y disposición para compartir su conocimiento.
Contenido

  • Aspectos generales de Excel.
  • Atajos y uso de teclado para el manejo de archivos grandes.

  • Aplicación de las fórmulas más comunes.
  • Uso de formatos de celda para resaltar y centrar información.

  • La gestión de bases de datos para generar de forma ágil la gestión de reportes.
  • Aplicación de algunas funciones condicionales para la generación de reportes.

  • Anidamiento o concatenación de funciones para la obtención de resultados específicos.
  • Gestión de bases de datos con el uso de las funciones lógicas.

  • Anidamiento o concatenación de funciones para la obtención de resultados específicos.
  • Construcción de funciones anidadas para casos específicos.
  • Gestión de bases de datos con el uso de funciones lógicas.

  • Construcción de tablas a partir de rangos de información.
  • El uso y aplicación de los filtros.
  • Uso de formularios para la introducción de datos y la inclusión de nuevos elementos en la cinta de opciones.

  • Introducción al manejo de tablas dinámicas.
  • Sus diferentes presentaciones a partir de la combinación de datos.
  • La gestión de bases de datos mediante el uso de tablas dinámicas.
  • Los diferentes formatos y combinaciones de campos de información.

  • Referencias relativas, absolutas y mixtas para la generación de reportes y la gestión de bases de datos.

  • El uso de texto en columnas y el uso de formatos para la presentación de informes.
  • El uso de formatos condicionales y la separación de texto de una celda para la gestión de bases de datos en forma eficiente.

  • El uso de texto en columnas y la concatenación de datos de diferentes celdas para la gestión de texto.
  • Construcción de fórmulas con funciones anidadas
