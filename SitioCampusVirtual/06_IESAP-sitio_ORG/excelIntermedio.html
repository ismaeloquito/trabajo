Descripción

Curso totalmente practico, enfocado en el manejo y aplicación de las diferentes herramientas de Excel, para desarrollar de manera más eficiente actividades relacionadas con la gestión, análisis de datos y presentación de reportes.

Objetivo general

En este curso aprenderás a construir funciones anidadas para generar reportes y gráficos específicos para el análisis de datos y aplicar eficientemente, en tu centro de trabajo, las herramientas de Excel para organizar, analizar, administrar, compartir información y automatizar la generación de reportes.

Requisitos
  • El participante debe contar con una computadora con: Micrófono, Cámara y Excel, preferentemente con la versión 2016 o posterior.
  • Es deseable haber cursado Excel Básico. En caso contrario el participante será responsable de su nivelación, con la finalidad de aprovechar al máximo el curso.
  • Conocimientos básicos del uso de una computadora y de diversas herramientas en la web.
  • Es deseable que cuente con habilidades de comunicación mediante diversas herramientas en la web, como el correo electrónico; administración del tiempo, autorregulación del aprendizaje, apertura al cambio y disposición de compartir conocimiento.

Contenido

  • Construcción de un cuadro de mando o dashboard con elementos básicos.
  • Revisión y aplicación de tablas y gráficas dinámicas.
  • Uso de segmentadores para visualización de reportes específicos.

  • Construcción de un cuadro de mando o dashboard con línea de tiempo y mapa coroplético.
  • La aplicación de las tablas dinámicas, gráficas dinámicas y segmentadores.
  • La aplicación de las líneas de tiempo y los mapas coropléticos.

  • Construcción de un cuadro de mando o dashboard con la creación de registros a partir de datos aleatorios.
  • Creación de bases de datos a partir de la generación de números aleatorios con diversas distribuciones de probabilidad.

  • Construcción de diagramas de Gantt, herramienta fundamental en la administración de proyectos.
  • A partir del uso de gráficas de barras apiladas, se construirán diagramas de Gantt con la asignación de responsables para cada una de las actividades.

  • Construcción de gráficos combinados: histograma, polígono de frecuencias y ojiva.
  • Revisión y análisis de diferentes distribuciones de probabilidad.

  • Construcción de varios gráficos de columnas internas.
  • Función de incrustación de gráficos a partir de tablas homogéneas.
  • Gráficas no convencionales para la visualización de datos.
  • Inclusión de ejes secundarios y el arreglo de los gráficos en diferentes planos.

  • Construcción y análisis diversas macros grabadas.
  • Edición de código Visual Basic.
  • Ejemplos comunes de macros grabadas para agilizar la gestión de tareas repetitivas
  • Introducción a la modificación de código

  • Construcción y análisis de diversas macros básicas con el uso de VBA.
  • Cambios a partir de la edición del código.
  • Ejemplos de macros básicas con el uso del VBA.
  • Edición de código.

  • Construcción y análisis de diversas macros más elaboradas con el uso de VBA
  • Construcción de macro con el uso de una subrutina
  • Subrutina de conteo para la demostración de algunos teoremas de la estadística

  • Desarrollo de fórmulas anidadas y el uso de funciones no convencionales.
  • Funciones para la gestión de texto.
  • Obtención de resultados específicos de manera automática.
