Suma de tres fracciones

La suma de tres fracciones, 4⁄3  +  3⁄2  +  1⁄4 se puede resolver de las siguientes maneras:
Se suman dos de las fracciones mediante el procedimiento de suma de fracciones con distinto denominador y la fracción resultante se suma con la tercera fracción.

4⁄3  +  3⁄2  =  4(2)⁄3(2)  +  3(3)⁄2(3)  =  8⁄6  +  9⁄6  =  8 + 9⁄6  =  17⁄6

17⁄6  +  1⁄4  =  17(4)⁄6(4)  +  1(6)⁄4(6)  =  68⁄24  +  6⁄24  =  68 + 6⁄24  =  74⁄24

Se simplifica dividiendo entre dos, en este caso, el numerador y el denominador.
74 ÷ 2⁄24 ÷ 2  =  37⁄12
Por lo tanto, el resultado de  4⁄3  +  3⁄2  +  1⁄4  =  37⁄12
